# headc

headコマンドのクローン

## Description

--help
Usage: headc [-n lines | -c bytes] [file ...]
指定したファイルの行頭から10行の情報を表示

コマンドライン引数: ファイル
オプション: 
-n 任意の数字　<指定した数字分の行数の情報を表示>
-c 任意の数字　<指定した数字分のバイト数の情報を表示　*接尾辞のついた数字にも対応>
